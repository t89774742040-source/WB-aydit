# Contributing

Спасибо за интерес к проекту.

## Правила изменений

- Основная бизнес-логика находится в `wb_seo_auditor_v5.html`.
- Перед изменениями проверяйте, что не ломается:
  - загрузка карточки из API;
  - загрузка карточки из ручного JSON;
  - парсинг CSV/XLSX;
  - генерация названия и описания;
  - экспорт JSON-отчета.

## Стиль кода

- Используйте UTF-8 и отступ 2 пробела.
- Не добавляйте лишние зависимости без необходимости.
- Новые правила фильтрации выносите в константы.
- Избегайте формулировок, похожих на машинный текст, в конечной генерации.

## Pull Request checklist

- [ ] Нет утечек токенов/секретов.
- [ ] Не добавлены бинарные и временные файлы.
- [ ] Обновлен `CHANGELOG.md`.
- [ ] Проверены основные пользовательские сценарии.
