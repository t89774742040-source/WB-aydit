WB SEO-аудитор карточки v5

Версия с жёсткой фильтрацией запросов. Приложение не склеивает случайные характеристики в «ключи», отдельно показывает ВЧ, СЧ и НЧ, вычищает персонажей и стоп-слова на каждом этапе, а новое название и описание собирает только из разрешённого семантического ядра.

Отдельные блоки ВЧ / СЧ / НЧ Строгий стоп-лист по персонажам и брендам Человеческое текстовое SEO-описание Если WB API блокирует CORS в браузере, загрузи карточку через ручной JSON
