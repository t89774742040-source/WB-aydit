# WB SEO Auditor

Локальный инструмент для аудита карточек Wildberries:
- загружает карточку из WB API или из ручного JSON;
- принимает файл поисковых запросов (CSV/XLSX);
- чистит семантику от шума и запрещённых сущностей;
- генерирует сильное наименование и SEO-описание;
- показывает рекомендации по улучшению карточки и характеристик.

## Стек

- Один файл приложения: `wb_seo_auditor_v5.html`
- Чистый HTML/CSS/JavaScript (без сборки)
- Внешняя библиотека для XLSX: SheetJS CDN

## Быстрый старт

1. Скачайте/клонируйте репозиторий.
2. Откройте `wb_seo_auditor_v5.html` в браузере.
3. Заполните:
   - WB API token (или вставьте ручной JSON);
   - артикул/nmID/vendorCode;
   - файл запросов (CSV/XLSX);
   - при необходимости ручные ключи и стоп-слова.
4. Нажмите:
   - `Загрузить карточку`
   - `Проверить карточку`
   - `Сгенерировать решение`

## Возможности

- Фильтрация персонажей/франшиз/чужих брендов.
- Кластеризация запросов: ВЧ / СЧ / НЧ.
- Удаление шумных соседних кластеров.
- Контроль переспама в описании и ограничение повторов.
- Учет актуальных (не прошедших) праздников для рекомендаций.
- Рекомендации по доп. характеристикам в формате:
  `Поле -> Что выбрать -> Что убрать -> Почему`.

## Структура

- `wb_seo_auditor_v5.html` — основное приложение.
- `docs/` — документация по архитектуре, дорожной карте и правилам.
- `CHANGELOG.md` — журнал изменений.
- `CONTRIBUTING.md` — правила вклада.

## Ограничения

- При прямом запросе из браузера WB API может блокировать CORS.
  В этом случае используйте поле `Ручной JSON карточки`.
- Точность генерации напрямую зависит от качества и релевантности файла запросов.
